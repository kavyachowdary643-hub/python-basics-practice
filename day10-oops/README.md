# Student Management System
A simple Python OOP project that stores and displays student information.
## Features
- Create student objects
- Store name, age, and branch
- Display student details using methods
## Concepts Used
- Classes
- Objects
- Constructor (__init__)
- self keyword
- Methods
- Attributes
