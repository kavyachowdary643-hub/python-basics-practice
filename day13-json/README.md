# Student Data Storage Using JSON
A simple Python project that stores and reads student information using JSON.
## Features
- Save student data to a JSON file
- Read student data from a JSON file
- Display student information
## Concepts Used
- JSON
- Dictionaries
- File Handling
- json.dump()
- json.load()
